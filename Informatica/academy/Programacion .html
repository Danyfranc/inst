Programacion

Clase de Programacion

La programación es un proceso que se utiliza para diseñar e implementar un conjunto de instrucciones o secuencias para que una computadora pueda realizar una tarea específica o resolver un problema determinado. Estas instrucciones o secuencias se conocen como programas o software.

¿Que aprendemos de la clase de programacion?

Apredemos a cosas como:

Definir el problema Esto implica entender claramente lo que se espera que haga el programa.

Diseñar el programa Una vez que se entiende el problema, el siguiente paso es diseñar cómo el programa va a resolverlo. Esto puede implicar la creación de diagramas de flujo y pseudocódigo para representar el flujo de la lógica del programa.

Escribir el código Una vez que se ha diseñado el programa, el siguiente paso es codificar ese diseño en un lenguaje de programación específico como C++, VB, etc.

Probar el programa Después de escribir el código, es importante probarlo para asegurarse de que funciona como se espera. Esto implica la ejecución de varias pruebas y la solución de los problemas que puedan surgir.

Depurar y optimizar Si durante las pruebas se encuentran problemas, estos deben ser corregidos. Este proceso se conoce como depuración. Además, se puede realizar una optimización del código para hacerlo más eficiente.

Mantenimiento Una vez que el programa está en uso, puede requerir mantenimiento para agregar nuevas funciones o corregir problemas que surjan.

Lenguajes usados en la carrera

En el bachillerato usamos 2 lenguajes en especifico que son ; C++ y VB (Visual Basic) y a continuacion les explicare que son:

C++es un lenguaje de programación de propósito general, orientado a objetos, creado por Bjarne Stroustrup como una extensión del lenguaje de programación C. Fue desarrollado en los laboratorios Bell en los años 80 y sigue siendo uno de los lenguajes más utilizados en la actualidad, especialmente para software de alto rendimiento, sistemas operativos, navegadores web, motores de videojuegos, etc.

Visual Basic (VB), por otro lado, es un lenguaje de programación orientado a eventos, desarrollado por Microsoft para su plataforma Windows. Originalmente lanzado en 1991, Visual Basic fue diseñado para facilitar la creación de aplicaciones gráficas de una manera más intuitiva que otros lenguajes de la época. Permite a los programadores arrastrar y soltar elementos visuales en un formulario y agregar código a estos elementos para definir su comportamiento.
